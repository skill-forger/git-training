# Version Control System - Git
Git - the leading version control system, to manage code efficiently. 
This course covers repository management, branching, merging, and collaboration using GitHub.

## What you will learn
- Understand the required command. Recommend to understand the commands in the optional section
- Understand the standard Git workflow for developing new features and fixing bugs.
- Adhere to branch naming and commit message conventions
- Be familiar with common git workflows
